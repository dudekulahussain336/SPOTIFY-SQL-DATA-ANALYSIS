# SPOTIFY-SQL-DATA-ANALYSIS
The Spotify SQL Project uses SQL queries to explore and understand a typical music store’s data. The queries aim to answer important questions about Spotify’s operations, customer habits, and sales, offering useful insights that can help improve the business and increase profits.The data is stored in several tables within the database. Based on the queries, some of the tables include employee, invoice, customer, invoice_line, track, genre, artist, and album.
